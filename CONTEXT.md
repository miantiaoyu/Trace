# 领域词汇

- **ERP 来源记录**：ERP 业务表中代表一条订单/柜号业务关联的记录，是追踪任务发现新对象和回写来源关联的依据。它的稳定身份优先使用 ERP 记录 ID，不把柜号单独当作业务主键。
- **头程追踪记录**：针对一条 ERP 来源记录保存的船司查询当前快照。它包含最近一次查询得到的原始返回、统一业务摘要和查询状态。
- **当前快照**：头程追踪记录中截至最近一次有效查询的当前位置、运输状态、目的地和预计时间；它不是完整历史轨迹。
- **目的地已到达**：船司返回了明确的目的港到达、提柜、还空箱或交付完成等实际证据。仅有 ETA、预计事件或目的港字段不能单独证明已到达。
- **未到达待追踪**：头程追踪记录存在，但目的地已到达为否，且仍满足下一次查询条件的记录。
- **原始返回**：船司官网本次查询返回的未经业务字段改写的数据，保留船司特有结构，作为排查和重新解析依据。
- **统一摘要**：跨船司抽取的有限业务字段，包括出发时间、最终目的地、目的地 ETA、当前位置、当前状态和船舶信息；来源没有明确提供的值保持为空。
